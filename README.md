# Credit Card Spending Dashboard

## Project Overview
This project presents a comprehensive **Credit Card Spending Dashboard** designed to analyze customer spending habits, interest generation, and transaction patterns. The dashboard provides a high-level view of financial performance across different card categories (Silver, Gold, Platinum, Blue) and expense types, enabling stakeholders to make data-driven decisions regarding credit card product offerings.

## Key Performance Indicators (KPIs)
The dashboard highlights the following aggregate metrics at the top level:
* **Total Customers:** 7T (Total customer base size).
* **Total Interest Earned:** 7.84M (Total revenue generated from interest).
* **Card Categories:** 4 (Distinct tiers of credit cards analyzed).
* **Total Spending Amount:** 45M (Total volume of transactions processed).

## Visualizations & Features

### 1. Temporal Analysis
* **Interest Earned by Year, Quarter, Month:** A line chart tracking financial performance over the timeline (Jan 2023 – Dec 2023), identifying seasonal trends and peaks in interest earnings.

### 2. Categorical Breakdown
* **Total Spending with Interest Earned by Card Category:** A funnel chart displaying the hierarchy of card contribution.
    * **Top Contributor:** Silver Cards
    * **Lowest Contributor:** Blue Cards
* **Total Transaction by Qtr:** A pie chart showing the distribution of transaction volume across the four quarters of the year (Q1–Q4), revealing a relatively balanced distribution (~25% per quarter).

### 3. Expense & Usage Analysis
* **Total Transaction Amt and Interest Earned by Expense Type:** A combination bar and line chart breaking down spending into categories:
    * Bills (Highest volume)
    * Entertainment
    * Fuel
    * Grocery
    * Food
    * Travel
* **Transaction Mode Filters:** Slicers available to filter data by method: **Chip**, **Online**, and **Swipe**.

### 4. Performance Tracking
* **Interest Target:** A gauge chart visualising the current performance against the set target (Current: 775.96 vs Target: 4.79K).

## Filters & Interactivity
The dashboard allows users to slice and dice data using the following controls:
* **Card Category:** Filter by specific card tiers (Silver, Gold, Platinum, Blue).
* **Quarter (Qtr):** View data for specific time periods.
* **Transaction Method:** Toggle between Chip, Online, and Swipe transactions.

## Key Insights
* **Silver Cards** are the most lucrative category in terms of total spending and interest earned.
* **Bills** represent the largest expense category for customers, significantly outpacing Travel and Food.
* Spending remains consistent throughout the year, with minor fluctuations observed in the quarterly breakdown.

## Tools Used
* **Data Visualization:** Microsoft Power BI
